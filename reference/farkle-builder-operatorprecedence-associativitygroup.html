AssociativityGroup Type

Namespace: Farkle.Builder.OperatorPrecedence
Assembly: Farkle.dll
Base Type: obj

A group of symbols that have the same associativity and precedence. This class and its descendants accept arrays of objects that correspond to symbols.

Terminals correspond to their designtime Farkle, literals correspond to their content as a string and productions correspond to the object passed or returned from the WithPrecedence family of functions in production builders.

Using the same terminal with multiple designtime Farkles (such as the original and a renamed one) causes undefined behavior and will most likely result in errors.
